**状态**：本地交付 SCAFFOLD（脚本框架 + 实施步骤）。**实测 baseline 由业务方运维 3-4 周机器跑**。

## 为什么本地不能跑

P0.0b.3 任务卡 §3.3 要求：4-8 实例并发跑 TBC 全链 16 万块的采样窗口。需要：

| 资源 | 单实例 | 8 实例并发 | 本机条件 |
|------|-------|-----------|---------|
| 磁盘（block files + chainstate）| 100GB | 800GB | 369GB（不够 8 实例）|
| 内存 | 8GB | 64GB | 58GB（紧张）|
| CPU | 4 核 | 32 核 | 16 核（够）|
| 时间 | 3-4 周 | 3-4 周（并发）| auto session 不能跑 3 周 |
| **datadir 副本** | **必须独立**（不能用生产）| 8 个副本 | **0 个**（生产 datadir 不能动）|

**关键约束**（用户硬性要求）：
- 不能动生产节点 PID 4185870
- 不能动生产 datadir `~/.bitcoin`（4GB chainstate + block files）
- 不能跑 invalidateblock / reconsiderblock（会改变 datadir）

所以 baseline 必须由业务方在 dedicated 机器上跑（不在本 dev session）。

## 业务方运维实施步骤

### Step 1：准备独立 datadir 副本

在另一台机器（或本机生产节点 stop 1 小时窗口）做：

```bash
# 1. 安全停掉一个全量节点（如 backup 节点）
bitcoin-cli stop
sleep 30

# 2. 复制 datadir 到 baseline 机器
rsync -av --exclude=wallets/ /var/lib/bitcoin/datadir/ baseline-host:/var/lib/baseline/instance-0/

# 3. 启动 backup 节点（恢复服务）
systemctl start bitcoind

# 4. 在 baseline 机器上 fanout 到 8 个独立副本
for i in 0 1 2 3 4 5 6 7; do
    rsync -av /var/lib/baseline/instance-0/ /var/lib/baseline/instance-$i/
done
# 注：此时 8 个副本同步同一 chain tip
```

### Step 2：每个实例启动一个 bitcoind（端口偏移）

```bash
# instance 0：rpcport 9332, p2p 9333
# instance 1：rpcport 9342, p2p 9343
# ... instance N：rpcport 9332+N*10
for i in 0 1 2 3 4 5 6 7; do
    rpcport=$((9332 + i * 10))
    p2pport=$((9333 + i * 10))
    bitcoind -datadir=/var/lib/baseline/instance-$i \
             -rpcport=$rpcport -port=$p2pport \
             -rpcbind=127.0.0.1:$rpcport \
             -rpcallowip=127.0.0.1 \
             -rpcuser=baseline -rpcpassword=baseline_local \
             -txindex=1 -daemon
done
```

### Step 3：运行 baseline replay

```bash
TBC_TIP=$(bitcoin-cli -rpcport=9332 -rpcuser=baseline -rpcpassword=baseline_local getblockcount)

for i in 0 1 2 3 4 5 6 7; do
    rpcport=$((9332 + i * 10))
    python3 tools/baseline-replay.py \
        --datadir /var/lib/baseline/instance-$i \
        --rpcport $rpcport \
        --instance $i --total 8 \
        --tbc-tip $TBC_TIP \
        --output /var/lib/baseline/instance-$i.json &
done
wait
```

### Step 4：合并 + 校验

```bash
# 合并 8 实例输出
python3 tools/merge-baseline.py /var/lib/baseline/instance-*.json \
    > consensus-baseline-v2.6.1.json

# spot check 100 高度跨实例一致性
python3 tools/baseline-spot-check.py \
    --baseline consensus-baseline-v2.6.1.json \
    --reference reference-node.json \
    --sample 100
# 期望：100% 一致
```

### Step 5：归档

```bash

# 不进 git，由 docs/plans/spike-results/baseline-storage-pointer.md 记录路径
```
