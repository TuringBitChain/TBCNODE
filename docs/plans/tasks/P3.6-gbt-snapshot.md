# P3.6 GbtSnapshotProvider 单 refresh worker + 合并队列 + N4 chain 用 seqlock

**工时**：1-2 周  **owner**：TBD  **前置**：P1.1 P3.3  **阻塞**：P4.1 P5.5

## 1. 目标

实现 `GbtSnapshotProvider`：单 refresh worker + 合并队列（pending atomic 标志，连续触发只产生 1 次实际拷贝）+ N4 修复（chain 信息走 seqlock Capture，不反向取 cs_main）+ condvar 长轮询 15s（H4）+ 主动过滤已上链 tx（HIGH-2）。完整 detailed-design §7.3。
