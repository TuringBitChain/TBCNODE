## 1. 目标

实现 `CCoinsViewCache::BatchWrite`，用 libcuckoo `insert + update_fn fallback`（K2 修复）+ `erase_fn` 实现块级原子提交。配合 `unique_lock(batchWriteMtx)` 保证 ConnectBlock 期间外界看到的是"全旧 → 全新"切换。

## 3. 代码改造

完整代码见 `cs_main-refactor-detailed-design.md` §4 BatchWrite 修复（K2），核心：

```cpp
bool CCoinsViewCache::BatchWrite(CCoinsMap& mapCoins, const uint256& hashBlockIn) {
    std::unique_lock bw(batchWriteMtx);
    {
        std::unique_lock m(metaMtx);
        hashBlock = hashBlockIn;
    }
    for (auto& [outpoint, entry] : mapCoins) {
        if (!(entry.flags & CCoinsCacheEntry::DIRTY)) continue;
        if (entry.coin.IsSpent()) {
            cacheCoins.erase_fn(outpoint, [&](CCoinsCacheEntry& e) noexcept -> bool {
                cachedCoinsUsage.fetch_sub(memusage::DynamicUsage(e));
                return true;
            });
        } else {
            size_t new_usage = memusage::DynamicUsage(entry);
            CCoinsCacheEntry copy = entry;
            bool ins = cacheCoins.insert(outpoint, std::move(copy));
            if (ins) cachedCoinsUsage.fetch_add(new_usage);
            else cacheCoins.update_fn(outpoint, [&](CCoinsCacheEntry& e) noexcept {
                cachedCoinsUsage.fetch_sub(memusage::DynamicUsage(e));
                e = std::move(entry);
                cachedCoinsUsage.fetch_add(new_usage);
            });
        }
    }
    mapCoins.clear();
    return true;
}
```
