# P1.4 TxnValidation 13 处直接 chainActive 读 → snap

**工时**：1 周  **owner**：TBD  **前置**：P1.1 / P1.3  **阻塞**：P1.6 / P3.* / P4.1

## 1. 目标

`src/validation.cpp` 内 `TxnValidation` 函数体（约 1515-1750 行）13 处直接读 `chainActive.Tip()` / `chainActive.Height()` 全部改成读 worker 局部 `Chainstate::Snapshot`。这是 v2.6.1 §1.2 K3 修复——worker 不持 cs_main。

## 2. 影响文件

| 文件 | 范围 |
|------|------|
| src/validation.cpp | TxnValidation 函数体 + 间接调用方 |
| src/test/txnvalidation_snap_tests.cpp | 新增 |

## 3. 代码改造

```cpp
// 旧
chainActive.Tip()->GetBlockHash()
chainActive.Tip()->nHeight
chainActive.Height()
GetMedianTimePast(chainActive.Tip())

// 新（worker 入口拍 snap，全函数体内只用 snap）
auto snap = GetChainstate().Capture();
snap.tip_hash
snap.height
snap.height
snap.mtp
```

13 处 + 间接 200+ 处统一替换。
