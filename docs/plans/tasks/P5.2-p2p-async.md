# P5.2 P2P SubmitAsync 集成 net_processing

**工时**：1 周  **owner**：TBD  **前置**：P3.3  **阻塞**：P5.3

## 1. 目标

`src/net/net_processing.cpp` 处理 P2P TX 消息时改走 `GetDispatcher().SubmitAsync(tx)`，不再用 PTV 老路径。

## 2. 影响文件

| 文件 | 改动 |
|------|------|
| src/net/net_processing.cpp | ProcessTxMessage 改 SubmitAsync |
| src/test/p2p_async_submit_tests.cpp | 新增 |

## 3. 代码改造

```cpp
void ProcessTxMessage(CNode* pfrom, ...) {
    TxRef tx = ReadTxFromMessage(...);
    GetDispatcher().SubmitAsync(tx);
    // 不再阻塞 net thread
}
```
