实现 `GetCoinConcurrent`：worker hot path 的并发安全 UTXO 读取。三级路径：L1 libcuckoo 命中 / L2 LRU 命中 / L3 LevelDB snapshot。完整代码见 detailed-design §1.3 K4 实现。

## 3. 代码改造

```cpp
bool CCoinsViewCache::GetCoinConcurrent(const COutPoint& outpoint, Coin& coin) const {
    std::shared_lock bw(batchWriteMtx);
    bool hit = cacheCoins.find_fn(outpoint, [&](const CCoinsCacheEntry& e) noexcept {
        coin = e.coin;
    });
    if (hit) return !coin.IsSpent();
    if (levelTwoCache.Get(outpoint, coin)) return !coin.IsSpent();
    Coin tmp;
    if (!base->GetCoin(outpoint, tmp)) return false;
    bool ins = cacheCoins.insert(outpoint, [&]() {
        CCoinsCacheEntry e; e.coin = tmp; e.flags = 0; return e;
    }());
    if (ins) cachedCoinsUsage.fetch_add(memusage::DynamicUsage(tmp));
    levelTwoCache.Put(outpoint, tmp);
    coin = std::move(tmp);
    return !coin.IsSpent();
}
```

LRU 容量 64MB（H3 修复 LevelDB 慢路径）。
