## 3. 代码改造

每个 functional test 覆盖 v2.6.1 核心场景：

- wallet：sendrawtransaction → mempool → wallet 最终一致；waitformempoolentry RPC
- zmq：hashtx / rawtx topic 顺序通过 SignalDispatcher
- REST：mempool 读 shared_lock 行为不变
- GBT：BatchWrite 期间 condvar 长轮询 15s
- reorg：6 块 reorg + 30 万 tx 涌入正确处理
- orphan：3 层 orphan chain 父 commit 后 trigger replay
- pruning：v2.6.1 后 pruning 行为不变
- indexer：txindex 信号顺序异步保证
