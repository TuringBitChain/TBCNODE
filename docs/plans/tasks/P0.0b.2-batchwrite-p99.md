## 1. 目标

在 prototype 阶段（实际 cacheCoins 改造在 P0，但本卡用 mock 量级）量化 BatchWrite 持锁时长，确认 10 万 UTXO 块 BatchWrite p99 ≤ 200ms。这是 v2.6.1 §2.3 trade-off 的核心 KPI——超过 200ms → BatchWrite 期间 worker 阻塞过长 → 平均 TPS 损失 > 1-2.5% → 整套方案不达标。

## 3. 代码改造

### 3.1 prototype benchmark

```cpp
// src/bench/batchwrite_bench.cpp
#include <bench/bench.h>
#include <libcuckoo/cuckoohash_map.hh>
#include <random>
#include <shared_mutex>

static void BatchWritePrototype(benchmark::State& state) {
    using Map = libcuckoo::cuckoohash_map<int64_t, std::array<uint8_t, 96>>;
    Map cache;

    constexpr size_t INITIAL = 50'000'000;   // 5000 万 UTXO baseline
    constexpr size_t BATCH = 100'000;        // 10 万 UTXO 块

    // 预填
    {
        std::array<uint8_t, 96> v{};
        for (size_t i = 0; i < INITIAL; i++) cache.insert(int64_t(i), v);
    }

    std::shared_mutex bw_mtx;
    std::mt19937_64 rng(42);

    while (state.KeepRunning()) {
        std::vector<std::pair<int64_t, std::array<uint8_t, 96>>> batch;
        batch.reserve(BATCH);
        for (size_t i = 0; i < BATCH; i++) {
            int64_t k = rng() % INITIAL;
            std::array<uint8_t, 96> v{};
            std::memcpy(v.data(), &k, sizeof(k));
            batch.emplace_back(k, v);
        }

        auto t0 = std::chrono::steady_clock::now();
        {
            std::unique_lock bw(bw_mtx);
            for (auto& [k, v] : batch) {
                if (rng() & 1) cache.erase(k);
                else cache.insert_or_assign(k, v);
            }
        }
        auto t1 = std::chrono::steady_clock::now();

        auto us = std::chrono::duration_cast<std::chrono::microseconds>(t1 - t0).count();
        state.SetIterationTime(double(us) / 1e6);
    }
}
BENCHMARK(BatchWritePrototype)->UseManualTime()->Iterations(1000);
```

## 7. 回滚条件
