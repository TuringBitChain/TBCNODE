# P3.7 SignalDispatcher per-tx FIFO（global_seq）

**工时**：1 周  **owner**：TBD  **前置**：P3.3  **阻塞**：P4.5b / P5.5

## 1. 目标

实现 `SignalDispatcher`：单线程 enqueue + 单线程 dispatch；global_seq atomic 保证 per-tx FIFO 顺序（H5）。订阅者（wallet/ZMQ/REST/index）通过此 dispatcher 接信号，避免 worker 同帧栈反向取锁（详 P4.5b）。完整 detailed-design §7.4。
