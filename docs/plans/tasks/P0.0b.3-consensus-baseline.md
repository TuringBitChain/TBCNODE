### 3.2 4-8 实例并发执行

```bash
# 启动脚本
NUM_INSTANCES=8
for i in $(seq 0 $((NUM_INSTANCES-1))); do
    python tools/baseline-replay.py \
        --instance $i \
        --total $NUM_INSTANCES \
        --datadir /data/baseline/instance-$i \
        --output /data/baseline/instance-$i.json &
done
wait

# 合并所有实例输出
python tools/merge-baseline.py /data/baseline/instance-*.json > consensus-baseline-v2.6.1.json
```

### 3.3 实例资源需求

| 资源 | 单实例 | 8 实例总 |
|------|-------|---------|
| 磁盘（block files + chainstate）| 100GB | 800GB |
| 内存 | 8GB | 64GB |
| CPU | 4 核 | 32 核 |
| 时间 | 3-4 周 | 3-4 周（并发）|

**风险**：8 实例需要 800GB 磁盘 + 64GB 内存机器。如果机器只 32GB / 256GB 磁盘 → 减到 4 实例，时间翻倍到 6-8 周（超 P0.0b 6 周预算）。需要先确认硬件资源。
