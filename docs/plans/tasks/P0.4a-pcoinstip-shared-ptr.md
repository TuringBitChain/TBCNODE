## 3. 代码改造

### 3.1 类型变更

```cpp
// src/validation.h:1094 改造前
// extern CCoinsViewCache *pcoinsTip;
// 改造后
extern std::shared_ptr<CCoinsViewCache> pcoinsTip;
```

### 3.2 init.cpp 启动序

```cpp
// src/init.cpp 启动
pcoinsTip = std::make_shared<CCoinsViewCache>(pcoinsdbview.get());

// src/init.cpp Shutdown 序（严格按 P-3 不变量）
void Shutdown() {
    // 1. 停 worker pool（drain in-flight）
    if (g_dispatcher) g_dispatcher->StopWorkers();
    // 2. 停 dispatcher
    if (g_dispatcher) g_dispatcher->Stop();
    // 3. 停 signal_dispatcher
    if (g_signal_dispatcher) g_signal_dispatcher->Stop();
    // 4. pcoinsTip 释放（shared_ptr 引用计数归零）
    pcoinsTip.reset();
    // 5. 然后才能调 UnloadBlockIndex（P-3 不变量）
    UnloadBlockIndex();
    // 6. pcoinsdbview / pblocktree
    pcoinsdbview.reset();
    pblocktree.reset();
}
```

### 3.3 worker hot path 持 shared_ptr 副本

```cpp
// PerChainWorker::ProcessItem 内
auto pcoins_local = pcoinsTip;   // 持 shared_ptr 局部副本
if (!pcoins_local) return;       // Shutdown 中 reset 后被检查
// 后续路径用 pcoins_local->GetCoinConcurrent 等
```
