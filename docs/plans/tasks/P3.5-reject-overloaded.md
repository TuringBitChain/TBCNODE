## 1. 目标

新增 `REJECT_OVERLOADED = 0x45`（F1 修复，不复用 0x44）；dispatcher 入口检测 `pcoinsTip->IsBatchWriteInProgress()` → 返回 REJECT_OVERLOADED。客户端按 503 语义重试。完整 detailed-design §8.1。

## 3. 代码改造

```cpp
// src/consensus/validation.h
static const uint8_t REJECT_OVERLOADED = 0x45;

// dispatcher
CValidationState ChainDispatcher::SubmitSync(TxRef tx) {
    if (pcoinsTip->IsBatchWriteInProgress()) {
        CValidationState s;
        s.Invalid(false, REJECT_OVERLOADED, "server-busy-batch-write");
        return s;
    }
    // ... 派发
}
```
