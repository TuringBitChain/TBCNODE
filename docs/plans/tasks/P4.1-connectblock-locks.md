# P4.1 ConnectBlock 提交阶段三锁原子提交（含 C-B 锁次序 + P-2 Flush abort）

**工时**：2 周  **owner**：TBD  **前置**：P0.2 P1.1 P3.3 P3.6  **阻塞**：P4.2-P4.6 / M2

## 1. 目标

改造 `src/validation.cpp` ConnectBlock 提交阶段：smtx unique → batchWriteMtx unique → **UpdateTip(seqlock)** → view.Flush() → RemoveForBlockNL（C-B 修复）；P-2 Flush 失败 std::abort + datadir 恢复说明。完整 detailed-design §5。

## 3. 代码改造

完整 detailed-design §5（v2.6 修订后版本）含：

```cpp
// 锁次序：UpdateTip 在 view.Flush() 之前（C-B）
GetChainstate().UpdateTip(pindex, GetBlockScriptFlags(...));
if (!view.Flush()) {
    LogPrintf("FATAL: view.Flush() failed after UpdateTip(%s)；abort\n", ...);
    std::abort();   // P-2 显式
}
g_mempool.RemoveForBlockNL(block.vtx, ...);
```
