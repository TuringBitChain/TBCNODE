## 1. 目标

worker ProcessItem 内 Resubmit 路径：race（tip/flags 变）retry 上限 10 + 指数退避 1ms-1s；hard（double-spend / missing input）retry 上限 1。超限走 RaceStash（N7 解耦 orphan_txns）。完整 detailed-design §6.2 + §2.3。

## 3. 代码改造

```cpp
// detailed-design §6.2 + §2.3 Resubmit
constexpr int MAX_RETRY_RACE = 10;
constexpr int MAX_RETRY_HARD = 1;
constexpr int BACKOFF_BASE_US = 1000;   // 1ms

void Resubmit(WorkItem&& item, ResubmitReason reason) {
    if (++item.retry_count > MAX_RETRY_RACE) {
        g_race_stash.Push(std::move(item.tx));
        if (item.promise) item.promise->set_value(REJECT_OVERLOADED, "max-retry");
        return;
    }
    // 指数退避
    int sleep_us = BACKOFF_BASE_US * (1 << std::min(item.retry_count, 10));
    std::this_thread::sleep_for(std::chrono::microseconds(sleep_us));
    // ... 其余 detailed-design §2.3
}
```
