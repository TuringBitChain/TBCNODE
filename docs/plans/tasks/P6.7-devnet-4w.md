## 1. 目标

**这是 v2.6.1 上线前的核心兜底**。

业务方澄清：TBC 开发网跟主网共识规则 100% 一样（仅 chainparams 字段 netMagic / fork heights / seeds 不同）。所以开发网 4 周稳定 = 真主网兼容性已验证。**不再需要 shadow node 4w + canary 4w + 渐进 4w**。

- 多节点开发网部署（至少 3 个 v2.6.1 节点 + 既有 v1 节点混跑）

- 4 周内任一指标偏离 → 立即关停 v2.6.1 + 启动 v1（datadir 兼容）
- 4 周稳定通过 → 进 P6.8 真主网部署
