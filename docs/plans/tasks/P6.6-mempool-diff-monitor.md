## 1. 目标

实现持续运行的 mempool diff 监控（M-K 双阈值告警）：

- **瞬态阈值**：60s 累计 vN 跟 reference 节点 mempool 差异 > 200 笔 → WARNING
- **持久阈值**：单 txid 在 vN 缺失但在 reference 存在 > 10 分钟 → ALERT

## 3. 代码改造

```python
# tools/mempool-diff-monitor.py
def main():
    last_seen = {}  # txid -> first_seen_us
    while True:
        vn_pool = set(rpc_call(VN_NODE, 'getrawmempool'))
        ref_pool = set(rpc_call(REF_NODE, 'getrawmempool'))
        diff = ref_pool - vn_pool   # ref 有但 vN 缺
        now = time.time()

        # 瞬态阈值
        if recent_60s_count(diff, last_seen) > 200:
            emit_warning("transient", count=...)

        # 持久阈值
        for txid in diff:
            if txid not in last_seen:
                last_seen[txid] = now
            elif now - last_seen[txid] > 600:
                emit_alert("persistent", txid=txid, age=now-last_seen[txid])

        # vN 入池后从 last_seen 删除
        for txid in vn_pool:
            last_seen.pop(txid, None)

        time.sleep(5)
```
