# P2.2 路由策略（投票 + power-of-two-choices + input cap）

**工时**：1-2 周  **owner**：TBD  **前置**：P2.1  **阻塞**：P2.3 / P3.*

## 1. 目标

实现 `FindWorkerForChain` + `PickLeastLoadedWorker` 路由策略：tx input 父在 inflight 表中找投票最多的 worker；找不到则用 power-of-two-choices 选最空闲。`MAX_INPUT_SCAN = 100` 防 DoS。完整代码 detailed-design §2.1。

## 3. 代码改造

完整代码见 detailed-design §2.1。

## 4. 验收 KPI

| KPI | 目标 |
|-----|------|
| 同子链路由命中率 | > 95%（input 父在 inflight 时） |
| MAX_INPUT_SCAN 防 DoS | 1000 input tx 路由耗时 < 50µs |
| power-of-two-choices 负载均衡 | 32 worker 标准差 < 平均值 20% |
| TSan 24h | 0 race |

## 5. 测试用例

```cpp
TEST(Routing, SameSubchainSameWorker) { /* 父子路由一致 */ }
TEST(Routing, IndependentChainBalanced) { /* 32 worker 分布均匀 */ }
TEST(Routing, DoSInputCap) { /* 10000 input tx 路由 < 50µs */ }
```
