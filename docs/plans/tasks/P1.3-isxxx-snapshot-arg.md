# P1.3 IsXxxEnabled / GetBlockScriptFlags 接受 Snapshot& 入参

**工时**：1 周  **owner**：TBD  **前置**：P1.1  **阻塞**：P1.4 / P3.* / P4.1

## 1. 目标

把 `IsGenesisEnabled / IsSchnorrMultisigEnabled / IsKycEnabled` 等所有 activation-height 检查函数添加 `(const Chainstate::Snapshot&)` 重载——保留老签名（接受 height 直接），新签名走 snap.height + snap.script_flags。

## 2. 影响文件

| 文件 | 范围 |
|------|------|
| src/validation.h / .cpp | 所有 IsXxxEnabled 加重载 |
| src/script/standard.cpp / .h | 类似改造 |
| src/script/interpreter.cpp | （仅检查使用点）|
| src/test/activation_height_tests.cpp | 新增 |

## 3. 代码改造

```cpp
// 旧
bool IsGenesisEnabled(const Config& config, int32_t nHeight);

// 新（保留旧签名，加重载）
bool IsGenesisEnabled(const Config& config, int32_t nHeight);   // 保留
inline bool IsGenesisEnabled(const Chainstate::Snapshot& snap) {
    return snap.height >= snap.genesisActivationHeight;
}

// GetBlockScriptFlags 类似
uint32_t GetBlockScriptFlags(const Config& config, const CBlockIndex* pindex);   // 保留
inline uint32_t GetBlockScriptFlags(const Chainstate::Snapshot& snap) {
    return snap.script_flags;
}
```
