## 3. 代码改造

```cpp
void ChainDispatcher::MarkCommitted(TxId txid) {
    // 现有 MarkCommitted 逻辑
    // 新增：trigger orphan replay
    auto children = g_orphan_txns->FindChildren(txid);
    for (auto& child_tx : children) {
        SubmitAsync(child_tx);
    }
}
```
