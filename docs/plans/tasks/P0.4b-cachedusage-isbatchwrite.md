# P0.4b cachedCoinsUsage atomic + IsBatchWriteInProgress + 1h 漂移

**工时**：1 周  **owner**：TBD  **前置**：P0.2 / P0.3  **阻塞**：P3.5（RPC busy 路径）/ M0

## 1. 目标

`cachedCoinsUsage` 改 `std::atomic<size_t>`，所有 BatchWrite / GetCoinConcurrent 路径 fetch_add/sub 一致。`IsBatchWriteInProgress()` 用 try_lock 判断（M-v3-2，无窗口）。1h 混合负载压测验证 0 漂移。

## 2. 影响文件

| 文件 | 改动 |
|------|------|
| src/coins.h | cachedCoinsUsage 改 atomic |
| src/coins.cpp | DynamicMemoryUsage / IsBatchWriteInProgress 实现 |
| src/test/coins_usage_drift_tests.cpp | 新增 1h 漂移测试 |

## 3. 代码改造

```cpp
// IsBatchWriteInProgress（无窗口实现）
bool CCoinsViewCache::IsBatchWriteInProgress() const {
    std::shared_lock bw(batchWriteMtx, std::try_to_lock);
    return !bw.owns_lock();
}

// DynamicMemoryUsage
size_t CCoinsViewCache::DynamicMemoryUsage() const {
    return cachedCoinsUsage.load(std::memory_order_relaxed);
}
```
