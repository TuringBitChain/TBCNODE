# P3.4 AsyncTrim 专用线程 + evict 拆批 1000 + yield

**工时**：0.5 周  **owner**：TBD  **前置**：P3.3  **阻塞**：P5.6

## 1. 目标

实现 `AsyncTrim` 专用线程：mempool 高水位触发，evict 拆批 1000 / 批，每批主动 unlock + yield 给 commit 路径机会。F8 trade-off 写实：每批 10ms × 10 批 = 累计 100ms，commit 短暂阻塞。完整 detailed-design §7.2。

## 3. 代码改造

完整 detailed-design §7.2（含 F8 修复：拆批 + yield）。

## 4. 验收 KPI

| KPI | 目标 |
|-----|------|
| evict 拆批 ≤ 1000 / 批 | 单元 |
| 每批主动 unlock 让 commit 抢锁 | 单元（commit 路径检测）|
| 单批持锁时长 | < 15ms |
| 100 万 mempool entries trim 完成 | < 30s |

